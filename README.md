## ✨ Features

- 🌌 **Particle engine** — animated canvas background with floating particles
- 🃏 **Card-based navigation** — full-screen slide transitions between sections
- ⌨️ **Typewriter effect** — dynamic role cycling animation
- 📜 **Certifications timeline** — modal image viewer for certificates
- 🏆 **Hackathon showcase** — AWS Nova AI Hackathon highlight card
- 📱 **Fully responsive** — mobile-first design with bottom tab navigation
- 🎨 **Glassmorphism UI** — frosted glass aesthetics with gradient accents
